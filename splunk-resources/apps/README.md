# Splunk Apps

This folder contains one subfolder per Splunk app discovered in the export.

Each app folder contains:
- `app.json` - App metadata
- `dashboards/classic/` - Simple XML dashboards
- `dashboards/studio/` - Dashboard Studio dashboards
- `saved-searches/` - Alerts and reports
- `macros/` - Search macros
- `lookups/` - CSV lookup tables
- `conf/` - Configuration files
